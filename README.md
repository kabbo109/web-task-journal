# Web Task Journal

This is a very easy project for beginners.

You can use this repository to:
- Write daily tasks
- Track simple learning
- Practice editing files
- Build a habit of GitHub commits

How to use:
1. Open any file
2. Add a small task or note
3. Save the file
4. Commit your change

Goal:
Complete something small every day.

Notes:
Write your daily tasks below.
